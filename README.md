## Getting Started

Clone down this repository. You will need `node.js` and `git` installed globally.


### Installation


1. Clone the repo
```sh
 git clone https://github.com/AnahiVera/Portfolio.git
```

2. Install packages
```sh
npm install
```
3. Run the project
```sh
npm start
```

4. Open in local host
```sh
http://localhost:3000
```
